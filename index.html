Инструменты веб - разработчика

Программы, сервисы и сайты необходимые для работы

Со временем, так или иначе, у каждого веб - разработчика накапливается огромное количество ссылок на ресурсы, сервисы, документацию и руководства которые нужны в работе. У меня такая же ситуация и поэтому решил все собрать и разложить по полочкам, для более удобного использования. Получился неплохой список, который пригодится всем, кому интересна веб - разработка.

P.S. список веб - сервисов периодически пополняется.

Редакторы кода

  • Notepad++

    Бесплатный редактор исходного кода и заметок, который поддерживает несколько языков.

  • Sublime Text 3

    Быстрый и удобный. Имеет мощный API Python, который позволяет плагинам расширять встроенные функции.

  • Brackets

    Современный текстовый редактор с открытым исходным кодом, который понимает веб-дизайн.

  • Netbeans

    Быстро и легко разрабатывать настольные, мобильные и веб-приложения.

  • Visual Studio Code

    Легкий, но мощный редактор исходного кода, который работает на вашем рабочем столе и доступен для Windows, MacOS и Linux.

  • Atom

    Текстовый редактор от GitHub.

  • Aptana Studio 3

    Мощный, с открытым исходным кодом редактор. Восстановлен с нуля и включает в себя новые функции, которые помогут вам быть более продуктивными.

  • Code Lobster

    Бесплатная межплатформенная IDE для разработки PHP / HTML / CSS / JavaScript.

  • Rapid PHP editor

    Быстрый и мощный редактор PHP-кода для Windows. Является самым полным программным обеспечением для кодирования PHP, HTML, CSS, JavaScript и других языков веб-разработки с инструментами для отладки, проверки, повторного использования, навигации и форматирования вашего кода.

  • PhpStorm

    Идеально подходит для работы с Symfony, Drupal, WordPress, Zend Framework, Laravel, Magento, Joomla, CakePHP, Yii и другими фреймворками.

Браузеры

  • Chrome

    Быстрый, удобный и безопасный браузер для всех ваших устройств.

  • Mozilla

    С удвоенной скоростью, встроенной защитой приватности. Новый Firefox - лучший способ веб-сёрфинга.

  • Opera

    Удобный, безопасный, быстрый браузер.

  • Microsoft Edge

    Более быстрый и безопасный браузер, разработанный для Windows 10.

Локальный сервер

  • Open Server

    Портативная серверная платформа и программная среда, созданная специально для веб-разработчиков с учётом их рекомендаций и пожеланий.

  • Denwer

    Джентльменский набор Web-разработчика. Локальный сервер (Apache, PHP, MySQL, Perl и т.д.) и программная оболочка, используемые Web-разработчиками для разработки сайтов на «домашней» (локальной) Windows-машине без необходимости выхода в Интернет. Главная особенность Денвера — удобство при удаленной работе сразу над несколькими независимыми проектами и возможность размещения на Flash-накопителе.

ПО для проектов

  • Node.js

    Среда выполнения JavaScript, встроенная в механизм JavaScript V8 для Chrome.

  • Gulp

    Инструментарий для автоматизации трудоемких задач в вашем рабочем процессе разработки.

  • Git

    Система управления распределенной версией с открытым исходным кодом, предназначенная для обработки от небольших до очень крупных проектов со скоростью и эффективностью.

  • Bower

    Менеджер пакетов для Интернета.

  • Koala

    Приложение для графического интерфейса для компиляции Less, Sass, Compass и CoffeeScript, чтобы помочь веб-разработчикам использовать их более эффективно. Koala может работать в windows, linux и mac.

  • Babel

    Компилятор JavaScript. Можно переводить код ES6 в ES5. После чего спокойно использовать в любом браузере.

  • Npm

    Менеджер пакетов для Node.js . Он был создан в 2009 году как проект с открытым исходным кодом, чтобы помочь разработчикам JavaScript легко обмениваться упакованными модулями кода.
